## 表单为何 阻止默认提交
- form action ?
  html 早期 收集用户的数据 默认由后端去处理，会重新刷新页面，是个不好的用户体验。
  页面会白一下。
  event.preventDefault() js雄起了 阻止默认提交,fetch(http) 去提交，没有必要刷新页面了。
  用户体验。
- 把用户当小白
  placeholder html5 新特性(提示用户)
  header 等语义化标签 html5
  requried关键字 表单必填项 减少一些用户犯错 
  - 函数模块化
    封装，复用，模块化
  - JSONView 插件必装 更好的查看API 接口数据
